Gutenberg block to render a template part inside a query loop

== Description ==

For use inside the Query loop, this block allows the rendering of a theme template part located in your child theme folder **wp-content/themes/child-theme/template-parts**


== Installation ==
1. Download and install plugin
2. Activate plugin

= Minimum Requirements =

* PHP 7.0 or greater is required
* MySQL 5.6 or greater is recommended
